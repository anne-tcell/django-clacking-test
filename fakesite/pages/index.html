This is a Django app to demo clickjacking.

Go here to test this site framing one of its own pages.

Go here to test a different site framing the same page as above: file://{{ BASE_DIR }}/fakesite/pages/frame.html

Install

0. Install Python 3.6. Other versions probably work too.
1. Clone this repository.
2. `cd django-clacking-test`
3. `pip install -r requirements.txt`
4. Create an app on `cd4.tcell-preview.io` (Admin > Applications).
5. Go to Admin > Download Agent > Server Agent, download a config file, and place it in the root directory of `django-clacking-test`.
6. Copy `fakesite/pages/clickjack-target.template.html` as `fakesite/pages/clickjack-target.html`.
7. Go to Admin > Download Agent > Javascript Agent, generate a key, and paste the script into `fakesite/pages/clickjack-target.html`.
8. `tcell_agent run python manage.py runserver`
9. Navigate to `localhost:8000`.

Modify clickjacking settings on the tCell dashboard at Settings > Policies > Clickjacking.
