If you opened this page from localhost:8000/frame, you're seeing the behavior when a site frames ones of it's own pages.

If you opened this file directly (i.e. from Finder on a Mac), you're seeing the behavior when a site frames a page from a site running tCell.

Open up the JavaScript console in Chrome for more information, e.g. refusals to frame the page based on it's Content Security Policy, depending on the settings you've chosen for tCell.
